nixApi
service in module nix.api

Description

Used to make calls to nutritionix api. Low level function to build api client on top of. All service methods are wrappers around service-function calls.

Usage

nixApi(endpoint, config);

Parameters

ParamTypeDetails
endpointstring

Relative api endpoint url e.g. '/estimated-nutrition/bulk'

configobject

Call-specific override for http calls configuration object https://docs.angularjs.org/api/ng/service/$http#usage The last override in chain of default object and nix.api.provider:nixApiProvider#methods_sethttpconfig to form final http call config.

Default object is built like that: 

{
    method:  'GET',
    url:     apiEndpoint + endpoint,
    headers: {
        'X-APP-ID':  credentials.appId,
        'X-APP-KEY': credentials.appKey
    },
    params:  {}
}
